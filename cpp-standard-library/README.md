
#### 简介
通过实现部分 C++ 标准库来学习 C++

#### 感悟
* 对于实现 C++ 标准库，只需明白原理，实现几个典型例子即可，否则就变成体力活了，得不偿失
* 对于普通程序员来说，对于模板元编程只需明白原理，实现能看懂即可
