
## 目录说明
天眼查爬虫

## 文件说明
```
001.py
    * 第一版爬虫程序
    * 已废弃
002.py
    * 第二版爬虫程序
    * 每次只爬取一年的一个省份数据, 避免被封
    * 遇到已经存在的年份或省份的数据, 跳过
    * 每爬取完一年的数据需要使用 003.sh 合并, 才能获取下一年的数据
003.sh
    * 将相同年份的数据合并成一个文件
004.py
    * 对 csv 文件补充编码信息
    * 已废弃
*.csv
    * 汇总后的数据文件
old
    * 临时数据或废弃数据
```

## 数据说明
满足以下条件的企业数
```
年份: 2010-2024
省份: 31个省直辖市
行业: 97个二级行业
登记状态: 存续
注册资本在5000万以上 -- 已删除
企业规模: 大型
```
