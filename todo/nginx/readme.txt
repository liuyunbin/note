





# HTML
## 超文本的标记语言
* 超文本: 指超链接
* 标记: 各种标签的类型
* 标签的默认效果不重要, 语义才重要

## 整体分类
* 块级标签
    * 每个块级标签独占一行, 多个块级标签从上往下排列
    * 默认宽度: 占满这个父级标签
    * 默认高度: 根据内容显示
    * 可以通过 CSS 调整宽和高
    * 块元素内几乎可以写任何标签
* 行内标签
    * 多个行内标签共享一行, 一行排满后, 下一行继续
    * 默认宽度: 根据内容显示
    * 默认高度: 根据内容显示
    * 不能通过 CSS 调整长和宽
    * 行内标签一般可以写行内元素, 不能写块级标签
* 行内块标签
    * 多个块级标签共享一行, 一行排满后, 下一行继续
    * 默认宽度: 根据内容显示
    * 默认高度: 根据内容显示
    * 可以通过 CSS 调整长和宽
    * 行内标签一般可以写行内元素, 不能写块级标签
* 块级标签 行内标签 行内块标签 可以通过 CSS 进行转化
* h1-h6 不能嵌套自身
* p 标签内不能包含块级标签
* a 标签可以嵌套除自身外的任何标签, 包括块级标签

# CSS
## 选择器
* 通配选择器: 主要用于清除样式
* 元素选择器: `a`
* 类选择器: `.class_name`, 类可以重复
* ID选择器: `#id_name`, ID 不能重复
* 交集选择器: `a.class_name`, 且, 一般不用 ID, 标签最多一个, 类目一般只有一个
* 并集选择器: `a,.class_name`, 或, 可以包含多个 元素 类 ID 选择器
* 后代选择器: `a .class_name`
* 子代选择器: `a>.class_name`, 子代所有后代的一种特例
* 相邻兄弟选择器: `a+.class_name`
* 通用兄弟选择器: `a~.class_name` 所有兄弟
* 属性选择器
    1. `[属性名]` ----------- 选中具有某个属性的元素
    2. `[属性名="值"]` ------ 选中包含某个属性, 且属性值等于指定值的元素
    3. `[属性名^="值"]` ----- 选中包含某个属性, 且属性值以指定的值开头的元素
    4. `[属性名$="值"]` ----- 选中包含某个属性, 且属性值以指定的值结尾的元素
    5. `[属性名*=“值”]` ----- 选择包含某个属性, 属性值包含指定值的元素
* 伪类选择器 -- 动态伪类
    1. :link ------ 超链接未被访问的状态
    2. :visited --- 超链接访问过的状态
    3. :hover ----- 鼠标悬停在元素上的状态
    4. :active ---- 元素激活的状态, 按下鼠标不松开
    5. :focus ----- 获取焦点的元素, 表单类点击, 触碰, 或使用 TAB, 正在输入
* 伪类选择器 -- 结构伪类
    1. :first-child ----- 所有兄弟元素中的第一个
    2. :last-child ------ 所有兄弟元素中的最后一个
    3. :nth-child(n) ---- 所有兄弟元素中的第 n 个, an+b
    4. :first-of-type --- 所有同类型兄弟元素中的第一个
    5. :last-of-type ---- 所有同类型兄弟元素中的最后一个
    6. :nth-of-type(n) -- 所有同类型兄弟元素中的 第n个
* 伪类选择器 -- 否定伪类
    1. not()
* 伪类选择器 -- UI伪类
    1. :checked ---- 被选中
    2. :enable ----- 可用的
    3. :disabled --- 不可用
* 伪类选择器 -- 目标伪类
    1. target
* 伪类选择器 -- 语言伪类
    1. lang
* 伪元素选择器
    * ::first-letter ----- 选中元素中的第一个文字
    * ::first-line ------- 选中元素中的第一行文字
    * ::selection -------- 选中被鼠标选中的内容
    * ::placeholder ------ 选中输入框的提示文字
    * ::before ----------- 在元素最开始的位置，创建一个子元素（必须用 content 属性指定内容）
    * ::after ------------ 在元素最后的位置，创建一个子元素（必须用 content 属性指定内容）

## 像素

* 文字: 指实际有多少像素点, 和文字的大小没有必然的关系

## 盒子模型
1. margin ---------- 外边距, 盒子与外界的距离
    * 上和左外边距影响自身的位置, 下和右外边距影响后面兄弟的位置
    * 行内元素的左右外边距没问题, 上下内边距不能设置
    * 块元素和行内块元素可以完美设置四个外边距
    * 块级元素左右外边距设置为 auto, 可以水平居中
    * 第一个元素的上外边距和最后一个元素的下外边距会作用在父元素上
        1. 父元素设置不为 0 的 padding
        2. 父元素设置不为 0 的 border
        3. 父元素设置 overflow:hidden
    * 上面兄弟的下边距和上面兄弟的上边距会合并, 取最大值, 修改: 只设置一个外边距
2. border ---------- 边框, 盒子的边框
3. padding --------- 内边距, 紧贴内容的补白区域
    * 行内元素的左右内边距没问题, 上下内边距不能完美设置
    * 块元素和行内块元素可以完美设置四个内边距
4. content --------- 内容, 元素中的文本或后代元素都是它的内容
    * 行内元素无法修改的宽和高
    * 块元素和行内块元素可以修改的宽和高

## 布局总结
* 水平对齐 ----- 指内容在父元素的内容区的对齐方式
    * text-align 只能配置在块元素, 不能配置在行内元素或行内块元素
    * 如果内容为文本, 父元素直接使用 text-align
    * 如果内容为行内元素, 父元素直接使用 text-align
    * 如果内容为行内块元素, 父元素直接使用 text-align
    * 如果内容为块元素
        * 左对齐: 直接在该块元素内设置, 右外边距为 auto, 默认
        * 居中对齐: 直接在该块元素内设置, 左右外边距为 auto
        * 右对齐: 直接在该块元素内设置, 左外边距为 auto
*  该元素在当前行的垂直对齐 vertical-align, 基线对齐, 顶部对齐, 中部对齐, 底部对齐
    * 在父标签上配置
    * 包括文本, 单元格, 以及其他行内元素和行内块元素
    * 加上 `font-size:0` 表示绝对居中, 不受字体大小的影响
* 垂直对齐 ----- 指内容在父元素的内容区的对齐方式
    * height 和 line-height 不能写在行内元素
    * 单行文本对齐
        * 顶部对齐: 默认
        * 中部对齐:  height = line-height
        * 底部对齐: 通过定位去做
    * 多行文本对齐
        * 顶部对齐: 默认
        * 中部对齐: 通过定位去做
        * 底部对齐: 通过定位去做
    * 块元素
        * 顶部对齐: 默认
        * 居中对齐: 设置 margin-top, (父元素内容区的高度-  子元素的高度) / 2
        * 底部对齐: 通过定位
    * 行内元素-行内块元素
        * 顶部对齐: 默认
        * 居中对齐: 设置: 父元素, height = line-height, 每个子元素, `verticalalign:middle;`
        * 底部对齐: 通过定位
* 行内元素和行内块元素可以被父元素当作单行的文本处理
* 元素之间的空白, 父元素设置: `font-size:0;`
* 行内块的幽灵空白: 基线对齐, 基线和底部有一定的空白
    1. 设置行内块的对齐方式
    2. display 修改为 block
    3. 设置父元素的 `font-size:0`

## 浮动
* 会脱离文档流
* 浮动后的任何元素的宽和高都由内容撑开
* 浮动后的任何元素都可以设置宽和高
* 浮动后的任何元素都和其他元素公用一行
* 没有任何的 margin 和 padding 的问题
* 四个方向的 margin 和 padding 都可以完美设置
* 不会有行内空白的问题
* 会对后面的兄弟元素有影响
* 父元素需要清除浮动

## 定位
* 相对定位
    * 相对于一开始的位置
    * 不会脱离文档流
    * 对其他元素的位置没影响, 可能会覆盖其他元素的值
* 绝对定位
    * 相对于包含块(第一个拥有定位属性的父元素)
    * 会脱离文档流
    * 所有元素宽高都被内容撑开, 都能设置宽高
* 固定定位
    * 相对于视口
    * 会脱离文档流
    * 所有元素宽高都被内容撑开, 都能设置宽高
* 粘性定位





* 定位元素使用的优先级高


## 伸缩盒模型

* flex-direction --- 主轴排列方向, 左右, 上下
* flex-wrap -------- 主轴布满后的换行方式
* flex-flow -------- 复合属性, 一般不用
* justify-content -- 主轴的对齐方式: 起点对齐, 终点对齐, 居中对齐, 均匀对齐
* align-items ------ 侧轴的对齐方式(单行)
* align-content ---- 侧轴的对齐方式(多行)
* flex-basis ------- 设置主轴的基准长度, -- 不常用
* flex-grow -------- 主轴有剩余空间时, 是否放大
* flex-shrink ------ 主轴空间不够时, 是否缩小
* flex ------------- 上三个属性的复合
* order ------------ 设置排序的顺序

* font-family ---------- 字体, 字体类型
* font-style ----------- 字体, 字体风格, 正常, 斜体
* font-weight ---------- 字体, 字体粗细, 细, 正常, 粗, 可以数字
* font ----------------- 字体, 复合写, 大小和族必须放到最后两位
* color ---------------- 文本, 颜色
* letter-spacing ------- 文本, 字母间距, 包括汉字, 可以是负值
* word-spacing --------- 文本, 单词间距(通过空格区分), 可以是负值
* text-decoration ------ 文本, 修饰符, 无, 上划线, 下划线, 删除线, 虚线, 波浪线, 颜色
* text-indent ---------- 文本, 首行缩进
* text-align ----------- 文本, 子元素在内容区的水平对齐方式
* line-height ---------- 文本, 行高, 可以控制行间距, normal, 像素, 倍数, 百分比相对于 font-size
    * 行在内容区的对齐方式, 默认顶部对齐
    * 单行文本, 如果 line-height 等于 height, 居中对齐
    * 多行文本, TODO
* vertical-align ------- 文本, 该元素在当前行的垂直对齐方式, 默认是顶部对齐, 不能处理块元素
* list-style-type ------ 列表, 设置列表符号
* list-style-position -- 列表, 设置列表符号的位置
* list-style-image ----- 列表, 自定义列表符号 url(图片地址)
* list-style ----------- 列表, 复合属性
* table-layout --------- 表格, 设置列宽度, 自适应, 还是等宽
* border-spacing ------- 表格, 单元格间距
* border-collapse ------ 表格, 合并单元格边框
* empty-cells ---------- 表格, 隐藏没有内容的单元
* caption-side --------- 表格, 设置表格标题位置
* background-color ----- 背景, 设置背景颜色
* background-image ----- 背景, 设置背景图片
* background-repeat ---- 背景, 设置背景重复方式
* background-position -- 背景, 设置背景图位置
* background ----------- 背景, 复合属性
* cursor --------------- 设置鼠标光标的样式
* display -------------- 隐藏, 或 转换元素类型
* width ---------------- 盒子内容区, 宽度
* max-width ------------ 盒子内容区, 最大宽度
* min-width ------------ 盒子内容区, 最小宽度
* height --------------- 盒子内容区, 高度
* max-height ----------- 盒子内容区, 最大高度
* min-height ----------- 盒子内容区, 最小高度
* padding-top ---------- 盒子内边距, 上内边距
* padding-right -------- 盒子内边距, 右内边距
* padding-bottom ------- 盒子内边距, 下内边距
* padding-left --------- 盒子内边距, 左内边
* padding -------------- 盒子内边距, 复合属性
* border-width --------- 盒子边框, 边框宽度
* border-color --------- 盒子边框, 边框颜色
* border-style --------- 盒子边框, 边框风格, none, 实线, 虚线, 点线, 双实线
* border --------------- 盒子边框, 边框复合属性
* border-left ---------- 盒子左边框, 同上
* border-left-style
* border-left-width
* border-left-color
* border-right --------- 盒子右边框, 同上
* border-right-style
* border-right-width
* border-right-color
* border-top ----------- 盒子上边框, 同上
* border-top-style
* border-top-width
* border-top-color
* border-bottom -------- 盒子下边框, 同上
* border-bottom-style
* border-bottom-width
* border-bottom-color
* margin-left ---------- 盒子外边距, 左外边距, 给块元素设置左右 auto , 可以水平居中
* margin-right --------- 盒子外边距, 右外边距
* margin-top ----------- 盒子外边距, 上外边距
* margin-bottom -------- 盒子外边距, 下外边距
* margin --------------- 盒子外边距, 复合属性
* overflow ------------- 溢出内容的处理方式, 显示, 隐藏, 滚动条
* overflow-x ----------- 水平方向溢出内容的处理方式
* overflow-y ----------- 垂直方向溢出内容的处理方式
* visibility ----------- 隐藏元素, 但位置还占着
* display -------------- 隐藏元素, 彻底隐藏
